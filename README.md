## üß† AI Matching Engine
The core matching algorithm calculates a **match score** based on:
| Factor | Weight |
|--------|---------|
| Topic Match | 50% |
| Availability Match | 25% |
| Experience Level Match | 15% |
| Skills Match | 10% |

> Strong matches (score > 80%) are shown as top 3 mentors for each student.
